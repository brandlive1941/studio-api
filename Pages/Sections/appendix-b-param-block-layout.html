Appendix B: Param Block Layout

Intrado provides the ability to integrate events with many different webcast and meeting providers. Provider-specific data is stored in the ParamBlock member of the event object. Logical fields are delimited within the ParamBlock with the caret (^). This allows for an unlimited number of configuration parameters to be defined. An example of this layout for 3 fields follows:

FieldValue1^FieldValue2^FieldValue3^

 

Below is a list of the provider-specific template type field layouts used within the ParamBlock.
