Locale ID reference

When creating or updating a user, it may be desirable to specify the user’s language and locale setting. The system uses the user’s Locale ID to determine what language to display when generating the user interface. Below is a table listing the valid values for Locale IDs. Please note that Notified is continuously adding new languages to the product, so contact support for an updated list of supported Language/Locale IDs.
